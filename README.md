# Лабораторная работа №5 — JDBC + PostgreSQL

## Предметная область

**«Золотое Яблоко» — управление каталогом товаров онлайн-магазина.**

Приложение позволяет управлять однотабличной базой данных товаров (products) через графический интерфейс на Java Swing, взаимодействуя с PostgreSQL через хранимые процедуры и функции


## Структура проекта

```
lab5/
├── src/main/java/lab5/
│   ├── db/
│   │   └── DatabaseManager.java   — подключение к БД, вызов хранимых процедур
│   ├── gui/
│   │   ├── ShopApp.java           — главное окно приложения (Swing)
│   │   └── ProductDialog.java     — диалог добавления/редактирования товара
│   └── model/
│       └── Product.java           — модель данных товара
├── src/main/resources/
│   └── init.sql                   — SQL-скрипт инициализации БД
└── lab5.iml                       — файл модуля IntelliJ IDEA
```

## Реализованный функционал

| Функция                            | Реализация                          |
|------------------------------------|-------------------------------------|
| Создание базы данных (`CREATE DATABASE`) | `DatabaseManager.createDatabase()` + `init.sql` |
| Удаление базы данных (`DROP DATABASE`)   | `DatabaseManager.dropDatabase()`   |
| Очистка таблицы                    | Процедура `sp_clear_table()`        |
| Добавление товара                  | Процедура `sp_add_product()`        |
| Поиск по бренду                    | Функция `fn_search_by_brand()`      |
| Обновление кортежа                 | Процедура `sp_update_product()`     |
| Удаление по бренду                 | Процедура `sp_delete_by_brand()`    |


Разграничение реализовано на уровне ролей PostgreSQL:

- **`shop_admin`** — полный доступ: `SELECT`, `INSERT`, `UPDATE`, `DELETE` на таблицу `products`, вызов всех процедур и функций.
- **`shop_guest`** — только чтение: `SELECT` на таблицу `products`, вызов `fn_get_all_products()` и `fn_search_by_brand()`.

При попытке гостя выполнить запрещённую операцию PostgreSQL возвращает ошибку `42501`, которая перехватывается и отображается как «Недостаточно прав доступа. Операция доступна только администратору.»

Все кнопки в GUI остаются видимыми для любого пользователя — ограничение реализовано через права БД, а не через скрытие компонентов интерфейса.

Администратор может создать нового пользователя БД с заданной ролью (admin/guest) через процедуру `sp_create_user()`. Новому пользователю назначается соответствующая роль (`shop_admin` или `shop_guest`).

## Хранимые процедуры и функции

| Имя                      | Тип       | Описание                                  |
|--------------------------|-----------|-------------------------------------------|
| `sp_create_table()`      | Процедура | Создание таблицы `products`               |
| `sp_drop_table()`        | Процедура | Удаление таблицы `products`               |
| `sp_clear_table()`       | Процедура | Очистка таблицы (`TRUNCATE`)              |
| `sp_add_product(...)`    | Процедура | Добавление нового товара                  |
| `sp_update_product(...)` | Процедура | Обновление существующего товара по `id`    |
| `sp_delete_by_brand(...)` | Процедура | Удаление всех товаров указанного бренда   |
| `sp_create_user(...)`    | Процедура | Создание пользователя БД с назначением роли |
| `fn_get_all_products()`  | Функция   | Получение всех товаров                    |
| `fn_search_by_brand(...)` | Функция  | Поиск товаров по бренду (`ILIKE`)         |

Из Java-кода (JDBC) вызываются только хранимые процедуры и функции. Исключения — `CREATE DATABASE` и `DROP DATABASE`, которые невозможно выполнить внутри PL/pgSQL из-за ограничений PostgreSQL (не могут выполняться внутри транзакции).

## Структура таблицы `products`

| Столбец    | Тип              | Описание             |
|------------|------------------|----------------------|
| `id`       | `SERIAL (PK)`    | Уникальный ID        |
| `name`     | `VARCHAR(255)`   | Название товара      |
| `brand`    | `VARCHAR(255)`   | Бренд                |
| `category` | `VARCHAR(100)`   | Категория            |
| `price`    | `NUMERIC(10,2)`  | Цена (руб.)          |
| `in_stock` | `BOOLEAN`        | Наличие на складе    |
